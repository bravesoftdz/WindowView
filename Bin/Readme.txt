软件名称：Window View Tool
中文名称：窗口查看器
版本：0.1
作者：ysai <ysai_cn@hotmail.com>

简介：
　　这是一个可以查看Windows窗口的工具，它可以查看窗口中组件的隶属关系，可以枚举窗
口的所有子组件，可以取得窗口的主菜单，并可对所有取得的对象进行简单操作（通过右键菜
单）。

中文语言包使用方法：
　　把.EXE和.INI文件解压到同一文件夹，运行后选择Option->Language->中文就可以使用中
文界面了。
